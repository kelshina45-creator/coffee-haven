# Coffee Haven

Coffee Haven is a modern responsive coffee shop website built with HTML, CSS and JavaScript.

The project includes a stylish landing page, a separate menu page, smooth animations, mobile navigation, menu filtering and a reservation form demo.

## Features

* Responsive design for desktop and mobile
* Modern glassmorphism navbar
* Smooth scroll navigation
* Animated sections on scroll
* Mobile burger menu
* Separate menu page
* Menu category filters: Coffee, Donuts, Croissants
* Dynamic menu item counter
* Reservation form with basic validation
* Dark premium coffee-inspired design
* Hover effects for buttons and cards

## Design Style

The website uses a dark coffee-inspired color palette with warm golden accents.
The design is focused on a premium coffee house atmosphere: soft contrast, elegant typography, glassmorphism elements and smooth interactive details.

## JavaScript Functionality

The JavaScript file includes:

* navbar style change on scroll
* mobile burger menu
* smooth scrolling for anchor links
* reveal animations on scroll
* active navigation link highlighting
* reservation form message
* menu filtering by category
* dynamic menu status counter
